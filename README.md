# ticket-medio-caixas
## 🧠 Lógica do Algoritmo

O algoritmo utiliza estruturas de repetição para:

1. Permitir múltiplos caixas
2. Registrar valores de compras até o usuário encerrar
3. Calcular médias individuais
4. Comparar resultados para encontrar o melhor desempenho

## 🔁 Estruturas utilizadas
- Enquanto (loop principal)
- Condicionais (Se/Senao)
- Acumuladores (soma)

## 📊 Complexidade
- Tempo: O(n), onde n é o total de clientes
